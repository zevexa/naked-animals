# Naked Animals Landing Page
Landing Page (The Odin Project)
This project is a solution to the Landing Page assignment from The Odin Project.

The goal of this assignment was to recreate a full webpage based on a provided design, using only HTML and CSS, with a strong focus on Flexbox layout techniques.

## 🚀 Overview

The page includes:
A hero section with navigation and call-to-action
An information section with card-style elements
A quote/testimonial section
A call-to-action (CTA) banner
A footer

The design fulfilled the brief by replicating the layout while utilizing the provided font details and color scheme. Additional content was added for the sake of my own entertainment and hopefully yours as well.

## 🛠️ Built With
* HTML5
* CSS3 (Flexbox)
* Google Fonts (Roboto)

## 🎯 Learning Objectives
This project helped reinforce:
Structuring a full webpage from scratch
Using Flexbox for layout and alignment
Managing spacing, sizing, and positioning
Applying consistent typography and color schemes
Translating a visual design into code

## 📝 Notes
This project is not responsive, as per the assignment requirements
Spacing and sizing are approximations, not pixel-perfect
Placeholder content is used where appropriate
